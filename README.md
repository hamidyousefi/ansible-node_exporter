# Node Exporter Ansible Role
## An Ansible role for setting up Node Exporter

### Usage
Install the role on your controller machine using the command below:

```bash
ansible-galaxy install hamidyousefi.node_exporter
```

Or add the following lines to your roles/requirements.yaml file. You can create this file if it does not exist:

```yaml
- name: hamidyousefi.node_exporter
  version: latest
```

You can find specific versions on the [Releases Page](https://github.com/hamidyousefi/ansible-node_exporter/releases)
